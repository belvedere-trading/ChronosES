def Chronos.Processor.BackgroundQueueProcessor.WaitForCompletion (   self,
  timeout = None 
)

Stops processing, then blocks the caller until all pending background operations have been completed.

Regardless of success, the process will be shut down upon completion of this method.

Parameters
timeoutthe number of seconds to block waiting for processing completion. If not supplied, this method will wait indefinitely for completion.
Returns
A bool representing whether or not all items were successfully processed. This can only be False if timeout is supplied.
